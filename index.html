About Bubble Tiles

Bubble tiles are curved-edge tiles made from circular arcs. Start with a regular polygon, then let each side bow outward or inward to create “bites” that fit together in surprisingly rich ways.

This app is an experimental companion for exploring bubble-tile geometry: build patches, snap compatible edges, group supertiles, and look for periodic, nonperiodic, and aesthetically interesting tilings.

Created by John Chase. Researchers, teachers, students, or makers doing work with bubble tiles are warmly invited to reach out: john.chase@mcpsmd.net.

Open the public app at johnrchase.github.io/bubble-tiles.

Physical bubble-tile materials and outreach inspiration are credited to Lauren Siegel and MathHappens Foundation. Visit MathHappens.org.

AI disclosure: ChatGPT 5.5 was used as a development assistant for this prototype. The mathematical ideas, research direction, and final editorial decisions remain John Chase’s responsibility.

Bubble Tiles 1.6 Help

Getting around
🧭
Guided Tutorial. About → Guided Tutorial offers a Quick Tool Tour and an interactive Explore tour with hands-on tiling challenges. The Explore tour saves your current canvas and lets you keep your tutorial work or restore the previous canvas when you leave.
∑
Bites and bumps. The lower-right status bar shows the total number of bites and bumps across bubble and puzzle-edge tiles currently on the canvas. For Soft Tiles, every point counts as two bites and every non-point counts as one bump, treating the shape as an H0 tile with T3 tiles attached. Polygon tiles, frames, and virtual lattice copies do not affect the totals.
🖥
Recommended device. Bubble Tiles works best on a desktop or laptop with a larger screen, mouse, and keyboard. Phone and tablet visitors receive a dismissible reminder when the app opens.
🖱
Pan. Hold Space and drag, or Ctrl/Cmd-drag anywhere. The pointer changes to a hand while either pan key is held; Ctrl/Cmd-drag is useful when the canvas is full of tiles.
Zoom. Use the mouse wheel or View → Zoom In / Zoom Out. View → Home / Reset View returns to the starting view. View → Full Screen enters or exits full-screen mode.
Adding and arranging tiles
▣
Tiles and Examples. Use the Tiles tab to add tiles. Open a family, then click a tile or drag it onto the canvas. “Add all” places a whole family in a reference arrangement. In the Examples tab, click an example's preview or name to replace the canvas and hover for its description. You can also drag a Bubble Tiles JSON file directly onto the app to open it.
◆
Penrose Kite and Dart. The Other tray includes the marked P2 Penrose prototiles, scaled for compatibility with the Penrose rhombs. Their long edges have length 1, their short edges have length 1/φ, and the green/red circular arcs show the matching rule: continue like-colored arcs across shared edges. Both edge lengths snap to other edges of the same length, and these polygon tiles are excluded from Bubble conversion and editing.
★
Stars and special polygons. The Other tray includes regular 5/2, 6/2, and 8/2 stellation outlines whose boundary segments all have length 1, a 45-45-90 triangle with side lengths 1, 1, and √2, its two-triangle tangram parallelogram, and a unit-edged 45-degree rhombus. The Regular Polygons tray's n-gon button opens a chooser for regular polygons with 3 to 120 sides.
💡
Context hints. The app sometimes shows a yellow note for mathematically important placements, including the Einstein-family monotiles, the One-of-each and Small frames, the chiral 3A / 3A* pair, and the Wide flower challenge. Add all suppresses individual tile notes while preserving notes about the collection as a whole. Every yellow note can be dismissed with its X button.
🧲
Snap. Snap is on by default. Drag or rotate tiles near compatible edges to see a gray preview; release to commit the snap. Equal edges snap directly, and a shorter edge can snap at each whole subdivision of a compatible longer edge—for example, a length-1 edge has four positions along a length-4 edge. Snapping also checks which side the tile interior is on, so concave pieces are less likely to snap into overlapping positions.
Overlap. Overlap is off by default. Turn it on to highlight likely collisions in red. Large layouts use focused checking so the app stays responsive.
Selection
☑
Select. Click selects one tile or group. Ctrl/Cmd-click or Shift-click toggles multiple selection. Drag on blank canvas to draw a box selection; no modifier key is required. Selection → All or Ctrl/Cmd+A selects all editable tiles.
Group / Ungroup. Group selected tiles with Selection → Group or G. Ungroup one level with Selection → Ungroup or U. Groups can be nested and can be used as supertiles.
Edit
↶
Undo / Redo. Use Edit → Undo / Redo, Ctrl/Cmd+Z, Ctrl/Cmd+Y, or Ctrl/Cmd+Shift+Z.
📋
Copy / Paste / Duplicate. Use Edit → Copy/Paste or Ctrl/Cmd+C and Ctrl/Cmd+V. When the app's internal clipboard contains copied tiles, Ctrl/Cmd+V pastes those tiles directly; otherwise it can accept an image from the system clipboard. Edit → Duplicate or D makes a nearby copy of the selected object. A duplicated multi-tile motif retains its internal layer order and is placed as one block above the original motif. These commands are also available by right-clicking a selected tile. Copies retain the original tile's scale, geometry, fill, and outline settings.
🖼️
Paste images. Copy a PNG, JPEG, WebP, or GIF image in another app, then press Ctrl/Cmd+V to place it at the center of the canvas. Pasted images can be selected, grouped, duplicated, layered, rotated, flipped, and scaled like tiles. They do not snap and cannot be converted or edited with the Bubble tools. Image data is embedded in saved JSON layouts, which can make layout files and share links much larger.
🗑
Delete / Clear Canvas. Edit → Delete removes the selected tiles or groups. Edit → Clear Canvas removes the editable layout and starts fresh.
Transform
⟲
Rotate. Set the degree amount in the Transform toolbar. Rotate CCW/CW uses that amount; Q and E use the same amount. The default is 15°.
Flip. Press H to flip horizontally or V to flip vertically. Flip H mirrors left/right across a vertical mirror line; Flip V mirrors up/down across a horizontal mirror line. These work on selected tiles, groups, and motifs.
Scale / Reset Scale. Select one or more tiles, then use the synchronized slider in either the Transform dropdown or Transform toolbar to set an absolute scale in 0.1 increments; it applies immediately and snaps around the 1× marker, where the moving thumb turns blue. For a custom value, type a decimal, percentage, fraction, or expression such as 3/2, sqrt(2)/2, or (1+sqrt(5))/2, then press Enter or leave the field. Expressions support parentheses, +, -, *, /, ^, and sqrt(); invalid expressions show a warning. Multiple tiles scale around the selection center. Transform → Reset Scale returns each selected tile to 1×. Snapping uses effective scaled edge lengths and recognizes whole-number subdivisions of compatible longer edges; Format Painter does not copy scale. When a complete lattice source motif is scaled together, its translation vectors scale with it.
Bubble
Arc Dual. Bubble → Arc Dual swaps every inward arc with an outward arc on selected bubble and puzzle-edge tiles. It also works on polygon-to-bubble tiles and custom arc variants.
Reverse One Arc. Select bubble or puzzle-edge tiles and choose Bubble → Reverse One Arc, or double-click a single editable bubble tile, then click an edge to reverse it. Matching shared edges in the current selection flip together; press Enter or click away to finish.
To Bubble Tile. Bubble → To Bubble Tile converts compatible polygon tiles into 60° arc-edge bubble tiles with all arcs initially pointing outward. Frames, Arc Lens, and the Einstein-family monotiles — Hat, Turtle, Spectre, Tile(1,1), and Stegosaurus Spectre — are excluded.
To Polygon. Bubble → To Polygon returns selected converted bubble or puzzle-edge tiles to straight polygon edges when possible.
Bubble Style. Choose 60° arc, puzzle, spike, or square-tooth edge profiles from Bubble → Bubble Style or the Bubble toolbar dropdown. It works on compatible bubble tiles; convert normal polygons first with To Bubble Tile. Arc Lens, soft tiles, and the Einstein-family monotiles are excluded. Reverse One Arc can flip individual styled edges inward or outward.
Arrange / Nudge
Layer order. Use Send to Bottom, Move Down, Move Up, and Bring to Top to control which selected tiles appear above or below others. The same options are available in the right-click menu. Duplicate and Tiling Fill preserve the source tiles' relative layer order.
⬅
Nudge. Arrow keys move the selected object. Shift+arrow makes a larger nudge.
Lattice
Tiling Fill. Select a motif, click Tiling Fill, drag a ghost copy to define vector A, then drag another ghost copy to define vector B. A source set can have only one lattice: select that same complete source set and choose Tiling Fill again to drag and readjust its A and B endpoints. Press Esc to cancel. Copies are virtual, preserve source layer order, and redraw as you pan and zoom.
▦×
Clear Lattice. Removes virtual lattice repeats while keeping the editable source tiles.
Locate. Centers the view on the editable parent/source tiles for existing lattice fills and flashes those parent tiles briefly.
Lattice opacity. The editable source motif stays full strength. Use the Opacity slider to control how light or dark the virtual lattice copies appear, including in Export Picture previews and exported pictures. Matching shared bubble-tile borders are also deduplicated between visible lattice copies and against the editable source tiles.
Style
Fill. Sets the selected tile's fill color, opacity, and pattern. Frames accept solid fill colors and opacity but not patterns. When one tile is selected, the controls and highlighted pattern preview reflect its current fill.
Outline. Sets the selected tile's border color, width, and line style. Canvas outlines scale naturally as the view or tile is scaled, use rounded joins, and deduplicate matching shared borders between neighboring tiles.
Format Painter. Select one source tile and activate Format Painter. Click a tile, group, or dragged selection to copy its style. Frame targets receive the permitted solid color, opacity, and outline settings without patterns, images, or text. The tool applies once and then turns off; press Escape to cancel.
Custom Image Fill. Select one non-frame tile and choose Custom Image from the Style menu or the image button on the Style toolbar. Choose, paste, or drop an image, or search Openverse for openly licensed media. Drag the image inside the tile and use the Zoom slider or mouse wheel over the preview to resize it. Openverse results retain source and license attribution. Custom images follow tile transforms and are preserved in copies, JSON layouts, share links, lattice fills, and picture exports.
Add Text. Select one non-frame tile and choose Add Text from the Style menu or the T button on the Style toolbar. Enter text, then choose its font, size, color, bold, and italic styling in the live preview. New text begins horizontal on the screen, then follows later tile rotations and reflections. Text is clipped to the tile and is preserved through copies, JSON layouts, lattice fills, Format Painter, and picture exports. Frames support solid-color fills rather than patterns, images, or text.
Canvas. Changes the canvas background color. Fill, Outline, and Canvas changes update immediately while preserving selection and overlap highlights.
Export and saving
Export Picture. File → Export Picture lets you drag a custom export rectangle, choose Free/Square/4:3/16:9 proportions, and export SVG, PNG, JPG, or PDF/print. Active lattice-fill copies visible in the chosen area are included at the selected lattice opacity.
💾
Save / Load. File → Save to Browser stores the current layout locally. File → Load from Browser restores it. JSON export/import is for archiving editable layouts and preserves the current view/zoom location; you can also drag a layout JSON file directly onto the canvas to open it. The bottom of the File menu lists the five most recently imported or exported JSON files from the current session; select one to reopen its cached layout.
🔗
Share Link. File → Copy Share Link copies a URL with the current canvas encoded after the #. When someone opens that URL, the app reconstructs the layout. This works best from the hosted GitHub Pages version; very large layouts may make links too long. If the canvas contains pasted images or custom image fills, the app warns that their complete data will be embedded in the URL and recommends Export JSON before allowing you to copy the link anyway.
