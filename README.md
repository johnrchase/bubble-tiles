# Bubble Tiles

An interactive web app for exploring bubble tiles, puzzle tiles, lattice fills, and related tiling patterns.

Created by John Chase.

## Use the app

The live app is hosted with GitHub Pages.

## Features

- Hexagonal, triangular, and square bubble tiles
- Puzzle-tile variants
- Lattice fill tools
- Arc dual transformation
- Styling, patterns, opacity, and food fills
- Save/load, JSON export/import, image export, and share links
